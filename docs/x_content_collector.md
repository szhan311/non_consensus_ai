## 收集技巧

1. **关注争论而非共识**
   - 两个人激烈辩论的 thread 往往最有价值
   - 找那些被 quote tweet 说"this is wrong"的帖子

2. **看评论区**
   - 高赞反对评论往往比原帖更有洞察
   - 找"I work in this field and..."开头的回复

3. **时间检验**
   - 6 个月前的预测现在应验了吗？
   - 找那些"我当时说 X，现在真的发生了"的 thread

4. **跨领域洞察**
   - 经济学家怎么看 AI？
   - 哲学家、社会学家、艺术家的观点
   - 非技术背景的人往往有独特视角
